Project Overview

This project explored common web vulnerabilities through guided exercises using OWASP Juice Shop. We simulated attacks such as IDOR (Insecure Direct Object References), broken access control, and weak encoding flaws like ROT13 decoding to access unauthorized pages.

As team leader, I guided the workflow for each vulnerability scenario, configured tools for cookie interception and JWT analysis, and led the report documentation. The project served as a practical reinforcement of secure coding and vulnerability testing techniques.
