Wireless & Mobile Programming Final Exam

Student enrollment application with authentication and credit limit system

Project Objective

Developed a full-stack mobile application for student enrollment with authentication, course selection, and credit limit enforcement (max 24 credits). The app solves the problem of manual course enrollment in academic communities.

Key Features

  • Student registration and login
  • Course catalog browsing
  • Credit limit enforcement (24 max)
  • Enrollment summary view
  • Input validation and error handling

Technical Details

Built with Android Studio using Java/Kotlin, connected to Firebase for authentication and data storage. Implemented responsive UI, data validation, and seamless navigation between enrollment screens.

Application Screens

Login Screen

Secure authentication with email/password validation. Shows "Invalid credentials" message for failed attempts.

Registration Screen

New users can create accounts with name, email, and password. Redirects to login after successful registration.

Home Dashboard

Welcome screen with navigation to enrollment features and personalized greeting message.

Course Catalog

Browse available courses with credit information. Enforces 24-credit maximum limit with validation.

Enrollment Summary

Displays selected courses and total credits. Provides confirmation of successful enrollment.
