Project Overview

This project involved building a web-based Security Risk Management System using the OCTAVE Allegro framework to assess and mitigate threats on critical assets. Users can input URLs for malware scans via the VirusTotal API, assess system vulnerabilities using the CIA model, and generate risk mitigation strategies.

The system categorizes risks by likelihood and impact, provides asset profiles, and recommends actions such as MFA, IDS, and data encryption. Results are logged and visualized in a real-time dashboard. The site supports role-based access and aligns with frameworks such as ISO 27001 and NIST 800-53.
